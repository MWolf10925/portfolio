# references/ — staging & moodboard (NOT part of the live site)

This folder is for gathering reference material and candidate assets **before** they (maybe) go into the production site. Nothing in here is served or deployed — it's a workspace.

## Folders to use

| Folder | What goes here |
|---|---|
| `current-site/` | Before screenshots of the current portfolio |
| `inspiration-sites/` | 3–5 sites you love (screen recordings preferred over screenshots for motion) |
| `motion-references/` | Short clips/gifs of animations whose *feel* you want |
| `brand/` | Your real logo source file (vector if you have it) |
| `screenshots/` | **Your** product screenshots (Instant Quote, etc.) — blur any client PII |
| `competitors/` | Other portfolios you want to beat |
| `textures/` | (Optional) CC0 textures you like |
| `3d-models/` | (Optional) CC0 3D you want to use |
| `candidate-assets/` | Stuff we're considering but haven't approved for the site |
| `generated-placeholders/` | AI-generated mood images — clearly labeled, never final, never fake proof |
| `license-notes/` | For every staged asset: source URL, license, creator, attribution |

> Create subfolders as you need them. They're intentionally not pre-created to avoid clutter.

## The two rules

1. **Inspiration ≠ production.** Reference sites/images guide mood, lighting, composition, and motion *feel* only. We never copy a design or reuse someone's image on the live site.
2. **Production assets must be yours, CC0, or properly licensed.** If you can't confirm an asset's license, it's inspiration-only. Log every staged/candidate asset's source + license in `license-notes/` and in `../ASSET_USAGE_MANIFEST.md`.

## Safe production-asset sources
Poly Haven (CC0), ambientCG (CC0), Sketchfab (CC0 filter), Unsplash/Pexels/Pixabay (verify per-image), Lucide/Phosphor/Heroicons/Tabler (icons), Google Fonts / Fontshare (fonts). See `../REFERENCE_LINKS_AND_SEARCH_PLAN.md` for links.

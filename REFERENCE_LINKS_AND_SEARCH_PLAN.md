## Recommended Moodboard Direction

Based on the audit: **"precision engineering, after dark."** Matte charcoal black, machined-metal neutrals, one confident **orange** accent, a single hero object (the **MW monogram**), faint **blueprint/schematic** geometry, controlled studio lighting, and **one signature scroll moment** (a line that morphs from a robot's path → a software pipeline → a shipped app). Cinematic but *restrained* — the content is the star; the motion frames it. Mostly **code-built** (zero licensing risk), with a CC0 HDRI/texture and Matt's real screenshots as the only external pieces.

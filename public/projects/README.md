# Project photos

Drop your build photos here, then they appear automatically in the featured rows.

Expected filenames (referenced in `src/data/projects.ts` → `image`):

- `robotics.jpg` — Robotics Autonomous Systems (a photo of the robot)
- `pid.jpg` — PID Tuning Workflow (a screenshot of the tuner / brain screen)
- `crm.jpg` — CRM Importer (a screenshot of the extension)

Any web format works (`.jpg`, `.png`, `.webp`) — just update the path in
`projects.ts` if you use a different extension. Recommended size: ~1600×1000.
Until a file exists, a branded placeholder frame shows in its place.
